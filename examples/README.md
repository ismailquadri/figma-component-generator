# Figma Component Generator Examples

This directory contains example projects demonstrating components generated by the Figma Component Generator.

## React Demo

A simple React application showcasing a Button component generated from Figma designs.

### Running the React Demo

```bash
cd react-demo
npm install
npm start
```

The demo will open at `http://localhost:5173`

### What's Included

- **Button Component**: A sample generated component with:
  - Multiple variants (primary, secondary, ghost)
  - Multiple sizes (small, medium, large)
  - Accessibility attributes (ARIA, keyboard navigation)
  - Focus states for keyboard users
  - Disabled state handling

### Features Demonstrated

- Component variants and sizes
- Accessibility features
- Event handling
- State management
- Responsive design

## Usage

These examples show what you can expect when generating components with:

```bash
generate-component generate --url "your-figma-url" --name YourComponent --framework react
```

## Adding Your Own Examples

To add your own generated component:

1. Generate a component using the CLI
2. Copy the generated files to this example project
3. Import and use the component in App.jsx
4. Update the showcase as needed
